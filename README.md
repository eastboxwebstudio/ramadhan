# Ramadhan Companion (Mobile App)

Aplikasi mobile ringkas berasaskan **Expo React Native** untuk bulan Ramadhan.

## Ciri-ciri

- Paparan waktu solat harian mengikut bandar/negara.
- Waktu utama Ramadhan:
  - **Akhir sahur** (Subuh)
  - **Berbuka** (Maghrib)
- Koleksi doa Ramadhan (Arab, transliterasi, maksud).
- Tip amalan & gaya hidup sihat sepanjang Ramadhan.

Data waktu solat diambil daripada API Aladhan.

## Jalankan aplikasi

```bash
npm install
npm run start
```

Kemudian buka melalui Expo Go (Android/iOS) atau jalankan pada emulator.

## Nota

- Anda boleh tukar lokasi dengan mengubah medan bandar dan negara pada aplikasi.
- Jika data lokasi tidak ditemui, aplikasi akan paparkan mesej ralat.
